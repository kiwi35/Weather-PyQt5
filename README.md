# PyQt Weather App

A Python desktop application built with PyQt5 that fetches real-time weather data for a user-entered city using the OpenWeatherMap API.

## Requirements
- Python 3
- PyQt5
- requests

## Setup
1. Create an API key at openweathermap.org.
2. Set the API key as an environment variable named OPENWEATHER_API_KEY.
3. Run the Python file.
